🔴 The Problem It Solves
Municipal response teams and city officials are often overwhelmed by fragmented incoming data:
Inefficient Reporting Channels: Traditional phone hotlines or disjointed web portals result in long delays, unverified reporting, and manual classification fatigue.
Duplicate Report Clutter: Major public hazards (e.g., severe potholes, water main breaks, blackouts) generate hundreds of identical individual citizen complaints, clogging the database and slowing dispatch times.
Delayed Disaster Awareness: Crisis events (e.g., sudden floods, electrical fires, structural blockages) hit social and local news media before official systems log them, leaving emergency responders blind to real-time situations.
🔵 The Solution: Community Hero
Community Hero creates a unified dashboard that automates detection, deduplication, and routing using artificial intelligence:
Citizen Empowerment Portal: Users report issues easily, upload images, tag severity, and trace real-time resolution status.
Autonomous AI Intelligence Module: A background agent monitors configured RSS news outlets, running them through LLM parsers to identify incidents before they are officially reported.
Intellectual Spatial Deduplication: The engine clusters issues using Jaccard text similarity paired with Haversine distance equations, ensuring no duplicate noise ruins coordinator workflow.
Interactive Disaster Heatmaps & Feeds: Beautifully rendered layouts track critical events dynamically, overlaying severity categories and confidence ratings on vector maps.
Animated AI Fallbacks: High-fidelity fallback vector graphics are generated programmatically for incidents without attached images, ensuring zero broken placeholders.
🛠️ How We Built It (Tech Stack)
Frontend: React 18 with TypeScript, Vite (optimized for lightning-fast speeds), and Tailwind CSS for responsive desktop-first precision.
Animations: Framer Motion (motion/react) driving tactile UI micro-animations and smooth layout transitions.
Database & ORM: PostgreSQL running on a production-ready relational architecture.
Core Mapping & Analytics: Lucide vector integrations, custom SVG-driven canvas modules, and Recharts-based telemetry charts.
AI Engine: Google Gemini SDK integration modeling smart categorizations, confidence rankings, spatial geocoding, and automated graphics fallback.
